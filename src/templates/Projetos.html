Criador-de-lista-de-tarefas | Projeto

A ideia para esse projeto veio com as primeiras aulas de Python no curso de Desenvolvimento de Software Multiplataforma. Toda lista de exercícios passada pelo professor deveria ser entregue por email em um arquivo .txt que contivesse os códigos de cada exercicio, porem, abrir cada arquivo de exercicio para dar Ctrl+C e Ctrl+V dentro de um arquivo de texto era algo entediante e repetitivo, por isso fiz esse programa para automatizar essa etapa.

TimeUp | Projeto

Esse foi o projeto de TCC do curso técnico em Desenvolvimento de Sistemas que fiz junto ao ensino médio. O projeto é um site que gera orçamentos de materiais de construção civil, as empresas se cadastrariam no site e colocariam seus produtos(contendo informações como tamanho, tipo, valor etc) a disposição no site e os usuários comuns poderiam se cadastrar e, utilizando o buscador de produtos do site, criar e salvar seus orçamentos.
